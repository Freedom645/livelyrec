## 7. 申し送り・観察事項

欠陥ではないが、テスト作成中に確認した事項を次工程へ申し送る。

1. **`recording_service.py` のスレッド駆動メインループ**（`_run` / `_connect_and_loop`、未カバー 58 行）は計画どおり結合テストの対象。OBS 実接続・再接続・指数バックオフの検証を結合テストで行う。
2. **`ocr/digit_template.py` の色相ラップアラウンド分岐**（`_color_mask` の `h_lo > h_hi`）は、現行の `JUDGE_COLOR` 定義（全色 `h_lo ≤ h_hi`）では実運用上未到達。将来 H=0 をまたぐ色帯を追加する場合に備えた防御実装であり、本工程ではカスタム `ColorRange` で論理のみ検証済。
3. **`master_service._select_chart` の `song.charts[0]` フォールバック**は、`Difficulty` 列挙が優先順位リストを完全に内包するため実質未到達。譜面 0 件時の `None` 返却は検証済。
4. 未カバー行の大半は、入力異常時の例外ハンドリング等の防御コード、および実機・実通信に依存する経路である。
5. 画像認識モジュールは合成画像で論理分岐を網羅した。実プレイ画像での認識精度は**システムテスト**（実プレイ録画によるエンドツーエンド検証）で確認する（R-013／R-022）。
